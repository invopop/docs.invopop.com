  <Accordion title="¿ What format does ARCA expect for periodic reports?">
    Periodic returns use ARCA's own form formats (F.2002, F.931, etc.) submitted via the SIAp/Aplicativo desktop tools or the online portal. There is no machine-readable spec equivalent to WSFE for these returns.
