  <Accordion title="How often must I submit myDATA reports?">
    Per document — issuance is real-time, classification is by the periodic return deadline (typically the 20th–26th of the following month). There is no batched periodic submission — myDATA is continuous.
  </Accordion>
  <Accordion title="How do I schedule periodic reports for Greece?">
    Φ2 returns are filed through Taxisnet by the taxpayer's accountant. Invopop captures myDATA-classified invoice data continuously — accountants reconcile against Invopop's exports when filing periodic returns.
  </Accordion>

  <Accordion title="What format does myDATA expect for periodic reports?">
    No periodic format — only per-document XML calls (`SendInvoices`, `SendClassifications`, `SendRevenueDoc`). Each is signed and submitted to IAPR's REST endpoint via the registered provider (ILYDA in Invopop's case).
  </Accordion>
  <Accordion title="What format do Greek periodic VAT returns use?">
    Φ2 and similar returns use Taxisnet's proprietary XML schemas, submitted through the portal. There is no public REST API equivalent to myDATA for periodic VAT declarations.
  </Accordion>
