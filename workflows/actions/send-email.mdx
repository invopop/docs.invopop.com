## Overview

"Send Email" will compose an email and attach files in the silo entry into it.

A common use case is generating a PDF in a previous step, and sending it to a customer.


## How to use

	<Step title="Step 1">

	</Step>

	</Step>
</Steps>

## Options

To
The recipient email

Automatically add invoice customers

BCC

Reply To:

Automatically add invoice supplier

Subject Template

Text Body Template

Combine attachments into a single zip file

Available template variables


| Variable               | Description               |
| ---------------------- | ------------------------- |
| `{{.full_code}}`       | Series and code combined  |
| `{{.series}}`   			 | Series                    |
| `{{.code}}`   				 | Code                      |
| `{{.issue_date}}`      | Issue Date                |
| `{{.supplier_name}}`   | Supplier name             |
| `{{.customer_name}}`   | Customer name             |
