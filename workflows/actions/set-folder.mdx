## Overview

Update the folder of a silo entry.

## Availability

* Invoice workflows
* Parties workflows
* Receipts workflows
* Rescue workflows

## How to use

This workflow action has a single dropdown labelled "Folder". This dropdown will list the folders and subfolders available to store your silo entries.
