## Timeline

Current and upcoming regulation for Argentina.

<Update label="Now" tags={["B2B", "B2G", "B2C"]}>
  **E-invoicing and e-reporting systems enforced to date**<br />
  Existing systems enforced in Argentina regarding e-invoicing, e-reporting and fiscalization.

  | System        | Description  | Invopop Coverage |
  |---------------|---------------|---------------|
  | [Factura Electrónica](#factura-electronica) | Argentina's mandatory electronic invoicing system, used to record, validate, and report all taxable transactions to ARCA in real time through a clearance model. | [Yes](/apps/argentina) |

</Update>

<Update label="1 Jun 2026" tags={["B2B", "B2C", "B2G"]}>
  **Real-time invoice issuance using C.A.E.**<br />
  [Mandate 5782 / 2025](https://www.argentina.gob.ar/normativa/nacional/resolución-5782-2025-419496/texto) comes into effect, with [CAE](https://servicioscf.afip.gob.ar/publico/comprobantes/cae.aspx) as the default real-time system. CAEA (the previous default batch reporting system) will only be used as a fallback when online systems are unavailable. The resolution now makes C.A.E. (Código de Autorización Electrónico) the default standard.
