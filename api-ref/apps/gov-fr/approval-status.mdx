---
title: "Get approval status"
openapi: "GET /apps/gov-fr/v1/entry/{silo_entry_id}/approval"
---

Returns the latest approval decision for the silo entry. A `204 No Content` means documents have been uploaded but no verdict has been recorded yet. A `200 OK` carries the decision: `approved: true` makes the party eligible for `directory.register`; `approved: false` includes a `notes` field with the reason for rejection.
