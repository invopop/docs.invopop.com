---
title: "Look up directory entries by SIREN"
openapi: "GET /apps/gov-fr/v1/directory/siren/{siren}"
---

Returns all entries from the local mirror of the PPF Annuaire that match the given SIREN. A SIREN may have multiple entries (one per registered SIRET / electronic address), so the response is always a list. An empty `results` array means the SIREN is not currently registered in the Annuaire.

Only available to live workspaces — sandbox enrollments receive `403 Forbidden`.
