The **Console** is the web interface of Invopop's API, providing a visual dashboard where you can manage documents, configure workflows and connect apps. It offers a user-friendly alternative to direct API calls, enabling you to create and process business documents, automate document handling processes, monitor workflow execution through job logs, and more.
