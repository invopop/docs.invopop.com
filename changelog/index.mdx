Recent changes shipped to the Invopop platform. For documentation updates, see the relevant guides directly.

<Update label="May 5, 2026" description="Week of April 27 – May 5">

## New

**France PA (Plateforme Agréée) is live.** Invopop is now [officially approved](https://www.impots.gouv.fr/je-consulte-la-liste-des-plateformes-agreees) as a Plateforme Agréée under the DGFiP mandate. You can now register parties in the Annuaire, send and receive invoices in UBL, CII, and Factur-X formats, and exchange lifecycle status updates (CDAR) over Peppol. See the [France PA guide](/guides/fr-pa) to get started.

**NO VERI\*FACTU support for Spain.** Spanish suppliers that don't operate under the VERI\*FACTU regime can now use a dedicated workflow path. Follow the [NO VERI\*FACTU supplier guide](/guides/es-noverifactu-supplier) and the [issuer guide](/guides/es-noverifactu).

**Peppol unregister workflow.** A new built-in workflow lets you revoke a Peppol participant's permission to issue and receive documents, disabling the party across SMP, SML, and Peppol.

**Argentina ARCA — Inicio de actividades.** Suppliers registering with ARCA can now declare a start-of-activities entry as part of the supplier registration flow. See the [ARCA suppliers guide](/guides/ar-arca-suppliers).

## Updates

**VERI\*FACTU supplier registration workflow.** The default registration workflow now verifies the Spanish Tax ID (NIF/CIF) up front and includes an unregister rescue path so a party can be cleanly disabled if a step fails. Existing workflows continue to work; new workspaces pick up the updated template automatically.

**Fetch all jobs endpoint.** The Transform API now exposes a [Fetch all jobs](/api-ref/transform/jobs/fetch-all-jobs) endpoint, making it easier to list and paginate through workflow jobs without querying them one by one.

**Chargebee email delivery simplified.** The Chargebee app no longer requires the business profile country to be set to Germany or Spain to forward generated PDFs (or X-Rechnung XML) to customers. Enable the Chargebee feature flag, mark customers with `cf_invopop_include`, and email notifications work everywhere. See the [Chargebee guide](/guides/chargebee).

## Removed

**InvoiceXpress integration retired.** The InvoiceXpress app and its associated workflows have been removed. Portuguese users should use the [AT Portugal app](/apps/at-portugal) directly — see the [AT Portugal guide](/guides/pt-at) for the recommended path.

</Update>
